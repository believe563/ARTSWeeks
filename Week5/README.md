### 问题20 有效的括号
**问题描述：**  
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。  

有效字符串需满足：  

左括号必须用相同类型的右括号闭合。  

左括号必须以正确的顺序闭合。  

注意空字符串可被认为是有效字符串。  

我的解法：Main.java中，由于字符种类较少，可以使用switch语句直接列出所有情况，详见Main.java中的解释
